### 前提処理

#### amplify cli

##### Install

```console
npm install -g @aws-amplify/cli@3.0.0
```

Permission Error で落ちる場合は下記を。
詳細は[こちら](https://qiita.com/noid11/items/6fb1660a2de108461840)から

```console
sudo npm install -g @aws-amplify/cli --unsafe-perm=true
```

##### 確認

```
amplify --version
```

version表示が出ればOK
